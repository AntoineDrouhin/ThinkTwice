# ThinkTwice

## Set Up

A la racine du projet :
```bash
cd api
npm install
node server.js
```

```bash
cd socket.io
npm install
node index.js
```
copier le repertoire `web` dans le dossier de votre serveur (apache ou autre)
```bash
cd web
bower install
```

Déplacez vous dans le répertoire mysql, lancez mysql puis :
```bash
source ProjetWeb.sql
```
